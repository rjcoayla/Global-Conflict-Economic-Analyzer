# 🌍 Observatorio Ciudadano: Guerra y Economía

**Monitor Abierto de Impacto Geopolítico en los Precios Globales**

Este proyecto es una herramienta de visualización e inteligencia de datos diseñada para rastrear la correlación entre conflictos bélicos regionales y la volatilidad en activos críticos como el **Petróleo Brent** y el **Oro (XAU)**. 

El objetivo es democratizar el acceso a la información estratégica, permitiendo que cualquier ciudadano pueda observar cómo la inestabilidad en zonas de tránsito marítimo o conflicto armado influye directamente en los indicadores económicos mundiales.

## 🚀 Funcionalidades Clave

* **Mapa Táctico en Tiempo Real:** Localización precisa de incidentes mediante coordenadas geográficas.
* **Monitor de Activos Críticos:** Precios actualizados de Petróleo y Oro.
* **Log de Ataques Regionales:** Historial interactivo con informes de impacto local y análisis técnico.
* **Análisis de Correlación:** Gráficos que comparan el historial de precios con los picos de tensión geopolítica.
* **Interfaz Adaptive-First:** Optimizado para una visualización clara tanto en computadoras como en dispositivos móviles.

## 🛠️ Tecnologías Utilizadas

* **Lenguajes:** HTML5, CSS3 (Tailwind CSS), JavaScript (ES6+).
* **Cartografía:** Leaflet.js (OpenStreetMap).
* **Gráficos:** Chart.js.
* **Iconografía:** FontAwesome 6.

## 📂 Requisitos de Instalación

Para desplegar este repositorio localmente o en un servidor, asegúrate de mantener la siguiente estructura de archivos:

```text
/
├── index.html          # Interfaz principal
├── data/
│   └── intel_report.json  # Feed de datos (Precios e Incidentes)
└── README.md           # Documentación



📊 Integración de Datos
El sistema está preparado para recibir datos de fuentes financieras y reportes de conflictos globales. El archivo intel_report.json debe contener los campos de precios actuales e incidentes geolocalizados para alimentar el mapa y los gráficos.
